# Sprint 6 - Terraform Associate Practice Exam

## 🎯 Objective
Test your Terraform knowledge with 60 practice questions similar to the HashiCorp Terraform Associate certification exam.

## 📋 Exam Structure
- **Total Questions**: 60
- **Time Limit**: 60 minutes (recommended)
- **Question Types**: Multiple Choice, Multiple Select, Fill-in-the-blank
- **Passing Score**: 70% (42/60 questions)

## 📚 Exam Domains Covered
1. **Understand Infrastructure as Code (IaC) concepts** (15%)
2. **Understand Terraform's purpose** (20%)
3. **Understand Terraform basics** (25%)
4. **Use the Terraform CLI** (30%)
5. **Interact with Terraform modules** (10%)

## 🚀 Instructions

### Quick Start
Run the exam runner script for an interactive experience:
```bash
cd day4-lab/05-exam-practice
./exam-runner.sh
```

### Manual Process
1. Review the questions in `questions.md`
2. Record your answers in `answer-sheet.md`
3. Check your results against `answer-key.md`
4. Review explanations for missed questions

### Files
- `exam-runner.sh` - Interactive exam script with timer
- `questions.md` - 60 practice questions
- `answer-sheet.md` - Template for recording answers
- `answer-key.md` - Correct answers with explanations
- `study-guide.md` - Key concepts and references

## ⏱️ Time Management
- **1 minute per question** average
- Mark difficult questions and return to them
- Review all answers before submitting

## 🎓 Study Resources
- [Terraform Documentation](https://registry.terraform.io/providers/hashicorp/azurerm/latest/docs)
- [Terraform Associate Exam Guide](https://developer.hashicorp.com/terraform/tutorials/certification)
- Hands-on experience from Sprints 1-5

## 📊 Scoring
Calculate your score:
```
Score = (Correct Answers / 60) × 100%
```

**Score Interpretation:**
- 90-100%: Excellent! Ready for certification
- 80-89%: Good understanding, review weak areas
- 70-79%: Passing level, additional study recommended  
- Below 70%: More practice needed

Good luck! 🍀
