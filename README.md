# Terraform Workshop - Day 4 Lab

## 🎯 Azure App Service Stack with Terraform

**Audience**: Trainees who already know Terraform fundamentals  
**Time-box**: ±5 hours, split into 6 sprints (+ bonus)  
**Workflow**: tick every task → commit → tag part-complete

## 🚀 Sprint Overview

### Sprint 1 — Remote Backend (⏱ ~30 min)
**Goal**: Create shared state in Azure Storage and initialize Terraform.

**Definition of Done** ✅
- `terraform init` prints "Successfully configured the backend 'azurerm'"
- Tests pass
- README generated by terraform-docs

### Sprint 2 — Network (⏱ ~45 min)
**Goal**: Build VNet foundation with NSG.

**Definition of Done** ✅
- `terraform plan` shows 5 resources
- Tests & docs pass

### Sprint 3 — Key Vault & IAM (⏱ ~45 min)
**Goal**: Provision secrets vault and access control.

**Definition of Done** ✅
- Key Vault alive, secret available
- Tests & docs pass

### Sprint 4 — App Service Plan & App Service + Hello World Deploy (⏱ ~70 min)
**Goal**: Deploy a live "Hello World" web app that securely reads its connection string from Key Vault.

**Definition of Done** ✅
- `terraform apply` creates/updates resources and deployment succeeds
- Browser shows "Hello World – DB OK"
- Tests & docs pass
- `terraform apply` creates 3 resources
- `curl <app-url>` returns HTTP 200
- Tests & docs pass

### Sprint 5 — SQL Database & Private Endpoint (⏱ ~50 min)
**Goal**: Azure SQL database reachable only via Private Link.

**Definition of Done** ✅
- Public access to port 1433 blocked
- `nslookup <db-fqdn>` resolves to 10.x address
- Tests & docs pass

### Sprint 6 — Terraform Associate Exam Practice (⏱ ~60 min)
**Goal**: Test your Terraform knowledge with 60 practice questions similar to the HashiCorp Terraform Associate certification exam.

**Definition of Done** ✅
- Complete all 60 practice questions
- Score ≥70% (42/60 questions) to pass
- Review explanations for missed questions
- Identify areas for additional study

### Bonus — Rename & Import (⏱ ~30 min)
**Definition of Done** ✅
- `terraform plan` shows only the rename (no recreation)
- NSG imported and changes can be applied

## 🛠️ Quick Commands

### Documentation
Generate terraform-docs for every module:
```bash
make docs
```

### Testing
Run all tests:
```bash
make test
```

### Infrastructure
```bash
make init    # Initialize all modules
make plan    # Plan all modules
make apply   # Apply all modules
make destroy # Destroy all modules (reverse order)
make clean   # Clean up temporary files
```
